# hanoi-tower

Les Tours de Hanoi sont un jeu de réflexion imaginé par le mathématicien français Édouard Lucas, et consistant à déplacer des disques de diamètres différents d'une tour de départ à une tour d'arrivée en passant par une tour intermédiaire, et ceci en un minimum de déplacements.  
Les disques sont empilés dans des tiges par ordre croissant (formant un petit cône).  
Il n’est possible de bouger qu’un disque à la fois. Seul un petit disque peut être placé sur un disque plus gros.  

## Comment lancer le projet

Pour lancer le projet, rendez vous sur le fichier graphic. Le programme se lance ensuite seul et montre les étapes de résolutions éffectué dans l'algorithme du fichier solve.
